# RICH Website - Frontend Repository 🌐

Welcome to the **RICH (Reach Into Cultural Heights)** website frontend repository!

## 🚀 Quick Start

### **Option 1: Shared Account** (Simple but needs 2FA)
- **Account**: `richincsad@gmail.com` (credentials in Notion)
- **2FA**: Contact Zarif for security codes (sent to iPad)
- **Access**: https://github.com/richinc-org/RICH

### **Option 2: Your Own Account** (Recommended) ⭐
- **Fork the repository** → Make changes → Send Pull Request
- **No 2FA issues** → Zarif reviews your work → Gets merged
- **Best for learning** and professional workflow

## 📁 What's Here

- **`docs/`** - 🌟 **MAIN FOLDER** - Website files (HTML, CSS, images)
- **`css/`** - Additional styling files
- **`RICH/`** - Main frontend application
- **`staging/`** - Backup and testing files

## ✏️ How to Edit (Option 2 - Recommended)

### **Step 1: Make Your Copy**
1. Go to: https://github.com/richinc-org/RICH
2. Click **"Fork"** button (top right)
3. Wait for it to finish

### **Step 2: Download & Edit**
1. Click **green "Code"** button → Copy URL
2. Open terminal and type:
   ```bash
   git clone [paste URL here]
   cd RICH
   ```
3. Edit files in the `docs/` folder
4. Save your changes

### **Step 3: Upload & Send for Review**
1. In terminal:
   ```bash
   git add .
   git commit -m "I updated the website"
   git push
   ```
2. Go to your fork → Click **"Compare & pull request"**
3. Write what you changed → Click **"Create pull request"**
4. Wait for Zarif to review and merge

## 🧪 Testing Your Changes

- **Test Site**: https://richinc-org.github.io/RICH/
- **Check**: Page loads, text looks correct, images display, links work
- **If wrong**: Don't panic! You can always fix it

2. **Upload**: CyberDuck → Connect to server → Upload files
